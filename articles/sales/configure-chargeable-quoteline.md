# <a name="configure-the-chargeable-components-of-a-project-based-quote-line"></a><span data-ttu-id="3ea88-103">Konfigurasikan komponen boleh dituntut bagi baris sebut harga berdasarkan projek</span><span class="sxs-lookup"><span data-stu-id="3ea88-103">Configure the chargeable components of a project-based quote line</span></span>

<span data-ttu-id="3ea88-104">_**Terpakai Kepada:** Project Operations untuk senario berasaskan sumber/bukan stok_</span><span class="sxs-lookup"><span data-stu-id="3ea88-104">_**Applies To:** Project Operations for resource/non-stocked based scenarios_</span></span>

<span data-ttu-id="3ea88-105">Baris sebut harga berdasarkan projek boleh mengandungi komponen dan komponen boleh dituntut.</span><span class="sxs-lookup"><span data-stu-id="3ea88-105">A project-based quote line can have included components and chargeable components.</span></span>

<span data-ttu-id="3ea88-106">Komponen yang disertakan adalah tertakluk pada kaedah pengebilan, pecahan pelanggan, had tidak boleh melebihi dan tetapan kekerapan invois yang ditakrifkan pada baris sebut harga berdasarkan projek.</span><span class="sxs-lookup"><span data-stu-id="3ea88-106">Included components are subject to the billing method, customer splits, not-to-exceed limits, and invoice frequency settings defined on the project-based quote line.</span></span>
<span data-ttu-id="3ea88-107">Subset komponen yang disertakan boleh ditandakan sebagai boleh dituntut dengan menggunakan **Jenis Pengebilan**.</span><span class="sxs-lookup"><span data-stu-id="3ea88-107">A subset of the included components can be marked as chargeable by using **Billing Type**.</span></span> <span data-ttu-id="3ea88-108">Anda boleh memilih salah satu daripada pilihan berikut dalam medan **Jenis Pengebilan** dalam konteks baris sebut harga:</span><span class="sxs-lookup"><span data-stu-id="3ea88-108">You can select one of the following options in the **Billing Type** field in the context of a quote line:</span></span>

   - <span data-ttu-id="3ea88-109">Boleh dituntut</span><span class="sxs-lookup"><span data-stu-id="3ea88-109">Chargeable</span></span>
   - <span data-ttu-id="3ea88-110">Tidak boleh dituntut</span><span class="sxs-lookup"><span data-stu-id="3ea88-110">Non-chargeable</span></span>

<span data-ttu-id="3ea88-111">Komponen boleh dituntut boleh ditakrifkan pada peranan dan kategori transaksi.</span><span class="sxs-lookup"><span data-stu-id="3ea88-111">Chargeable components can be defined on roles and transaction categories.</span></span>

<span data-ttu-id="3ea88-112">Kebolehtuntutan yang ditakrifkan pada peranan untuk baris sebut harga projek hanya diguna pakai untuk kelas transaksi **Masa**.</span><span class="sxs-lookup"><span data-stu-id="3ea88-112">Chargeability that is defined on roles for a project quote line only applies to the **Time** transaction class.</span></span> <span data-ttu-id="3ea88-113">Jika baris sebut harga projek mempunyai **Termasuk Masa** = **Tidak**, tab **Peranan Boleh Dituntut** tidak tersedia.</span><span class="sxs-lookup"><span data-stu-id="3ea88-113">If a project quote line has **Include Time** = **No**, the **Chargeable Roles** tab isn't available.</span></span>

<span data-ttu-id="3ea88-114">Kebolehtuntutan yang ditakrifkan pada kategori transaksi untuk baris sebut harga projek hanya diguna pakai untuk kelas transaksi **Perbelanjaan**.</span><span class="sxs-lookup"><span data-stu-id="3ea88-114">Chargeability that is defined on transaction categories for a project quote line only applies to the **Expense** transaction class.</span></span> <span data-ttu-id="3ea88-115">Jika baris sebut harga projek mempunyai **Termasuk Perbelanjaan** = **Tidak**, tab **Kategori Boleh Dituntut** tidak tersedia.</span><span class="sxs-lookup"><span data-stu-id="3ea88-115">If a project quote line has **Include Expenses** = **No**, the **Chargeable Categories** tab isn't available.</span></span>
